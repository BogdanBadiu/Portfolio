# Planets — OpenGL Solar System

A simple solar-system / planet scene rendered with **OpenGL** in C. Built for the *Computer Graphics* course.

## Features

- Multiple planets rendered with sphere primitives
- Orbital animation
- Lighting (sun as light source)

## Code

- `PLANET.C` — main scene (ANSI C, OpenGL fixed-function)

## Tech

C, OpenGL (fixed-function), GLUT, Visual Studio

## Run

Open `Proiect Planete.sln` in Visual Studio and run, or launch the prebuilt `Debug/Proiect Planete.exe` directly.
