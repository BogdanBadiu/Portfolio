# Ship v3.0 — OpenGL 3D Ship Scene

An animated 3D ship scene rendered with **OpenGL** in C++ with interactive ship and camera controls. Built for the *Computer Graphics* course.

## Features

- 3D ship model on water
- Interactive ship navigation (arrow keys)
- Free camera controls (WASD)
- Pause / options / reset commands

## Controls

| Key | Action |
|---|---|
| `↑` / `↓` / `←` / `→` | Move the ship |
| `W` / `A` / `S` / `D` | Camera movement |
| `P` | Pause |
| `O` | Options |
| `R` | Reset |

## Tech

C++, OpenGL (fixed-function), GLUT, Visual Studio

## Run

Open `Vapor v3.0.sln` in Visual Studio and run.
