# PokeBall — OpenGL Polygon Drawing

A Poké Ball rendered using basic **OpenGL** polygon primitives. Built for the *Computer Graphics* course as an introductory exercise on 2D/3D polygon rasterization and color filling.

## Code

- `Desenare_poligoane.c` — polygon-drawing logic (ANSI C)
- `PokeBallIco.ico` — application icon

## Tech

C, OpenGL (fixed-function), GLUT, Visual Studio

## Run

Open `PokeBall.sln` in Visual Studio and run.
