# Forest Scenery — OpenGL 3D Landscape

A real-time 3D forest scene rendered with **OpenGL** (legacy fixed-function pipeline) in C++. Built for the *Computer Graphics* course.

## Features

- 3D terrain with grass and ground textures
- Trees and forest props
- **Skybox** — six-face cube map for the sky
- Navigable camera
- Texture mapping with TGA assets

## Texture Assets (in `Debug/`)

- `pos_x.tga`, `neg_x.tga`, `pos_y.tga`, `neg_y.tga`, `pos_z.tga`, `neg_z.tga` — skybox faces
- `groundText.tga` — ground texture
- `grassMare.tga` — grass texture

## Tech

C++, OpenGL (fixed-function), GLUT, Visual Studio

## Run

Open `Proiect PeisajV2.2.sln` in Visual Studio and run, or launch the prebuilt `Debug/Proiect PeisajV2.2.exe` directly.
